# CardPhysics

Alternate `@main` entry point using the SwiftUI `App` protocol. Contains a single file `CardPhysicsApp.swift` that imports `CardPhysicsKit` and renders `CardPhysicsView()` in a `WindowGroup`.

This is a standalone entry point separate from the Xcode project's app target at `CardPhysicsApp/CardPhysicsApp/CardPhysicsAppApp.swift`.
