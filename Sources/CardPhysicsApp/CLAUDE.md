# Sources/CardPhysicsApp

SPM executable target defined by the root `Package.swift`. Contains a placeholder `@main` entry point that prints "Hello, world!".

This is **not** the real app entry point. The actual app is built via the Xcode project at `CardPhysicsApp/CardPhysicsApp.xcodeproj`, which uses `CardPhysicsApp/CardPhysicsApp/CardPhysicsAppApp.swift` as its `@main`.
